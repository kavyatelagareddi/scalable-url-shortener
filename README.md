🚀 Scalable URL Shortener (Distributed Systems)

A full-stack URL shortening platform built using React, FastAPI, PostgreSQL, Redis, and Docker.

The application allows users to generate short URLs, create custom aliases, set expiration dates, and track analytics efficiently.

✨ Features
🔗 URL shortening
🏷️ Custom aliases
⏳ Expiration dates
📊 Analytics tracking
⚡ Redis caching
🔄 Duplicate URL detection
🔢 Base62 encoding
🐳 Docker support
🛠️ Tech Stack
Frontend
React
Tailwind CSS
Backend
FastAPI
Database
PostgreSQL
Caching
Redis
Containerization
Docker
🏗️ Architecture
React Frontend
       ↓
FastAPI Backend
       ↓
Redis Cache
       ↓
PostgreSQL Database
🚀 Run Locally
Backend
cd backend

venv\Scripts\activate

uvicorn app.main:app --reload
Frontend
cd frontend

npm install

npm run dev
Docker
docker compose up
