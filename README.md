# 🚀 Scalable URL Shortener

A full-stack URL shortening platform built using React, FastAPI, PostgreSQL and Redis.

## Features

- URL shortening
- Base62 encoding
- Custom aliases
- URL expiration
- Analytics dashboard
- Duplicate URL detection
- Redis caching
- Modern React UI

## Tech Stack

Frontend

- React
- Tailwind CSS

Backend

- FastAPI

Database

- PostgreSQL

Caching

- Redis

Containerization

- Docker

## Architecture

React → FastAPI → Redis → PostgreSQL

## Run locally

Backend

```bash
uvicorn app.main:app --reload
```

Frontend

```bash
npm run dev
```

Docker

```bash
docker compose up
```
